# 🐉 Bestiary

Examen Ordinario — Aplicación web desarrollada con **React** y **Vite** que implementa un bestiario interactivo, permitiendo consultar y explorar criaturas/monstruos con sus características.

## 📋 Descripción

Este proyecto fue desarrollado como examen ordinario de la materia de Desarrollo Web. Consiste en una SPA (Single Page Application) que consume y muestra información sobre criaturas de forma dinámica utilizando componentes de React.

## 🚀 Tecnologías utilizadas

- **React** — Librería para construcción de interfaces de usuario
- **Vite** — Entorno de desarrollo y bundler ultrarrápido
- **JavaScript **
- **HTML / CSS**
